## 📌 Visão Geral
Este é um projeto em Python que implementa a clássica **Cifra de Vigenère**. O script funciona direto no terminal de comando (CLI) e permite que o usuário criptografe e descriptografe mensagens de forma interativa, além de oferecer um breve resumo educacional de como o algoritmo funciona.

A Cifra de Vigenère é um método de criptografia de texto que utiliza uma série de diferentes cifras de César baseadas em letras de uma senha (chave). 

## ⚙️ Funcionalidades
* **Explicação Interativa:** O programa oferece um resumo de como a cifra funciona para usuários leigos.
* **Criptografia:** Transforma um texto em claro em um texto cifrado usando uma chave fornecida pelo usuário.
* **Descriptografia:** Reverte o texto cifrado para o texto original usando a mesma chave.
* **Preservação de Formatação:** O algoritmo inteligentemente ignora espaços e caracteres especiais, aplicando a criptografia apenas nas letras (alfabeto) e mantendo a estrutura da frase intacta.
